# Faculty of Information and Communication Technology <br/> ITDS242 Web Technologies Lab <br/> Web Service Lab
## คำชี้แจง
- แบบฝึกปฏิบัตินี้มีทั้งหมด 1 ข้อ ที่เกี่ยวกับการเรียกใช้เว็บเซอร์วิสสาธารณะ โดยใช้จาวาสคริปต์
- การส่งงานขอให้นักศึกษาส่งงานผ่าน Github Classroom ภายในคาบถัดไป

แบบฝึกหัดภาคปฏิบัตินี้ นักศึกษาจะได้เข้าถึงเว็บเซอร์วิสสำหรับหาข้อมูลของอาหารจาก [spoonacular API](https://spoonacular.com/food-api) มีเว็บเซอร์วิสสำหรับการให้ข้อมูลเกี่ยวกับอาหาร อาทิ เครื่องปรุง ส่วนผสม และโภชนาการ เป็นต้น
นักศึกษาสามารถทำตามขั้นตอนต่อไปนี้
1. สมัครเพื่อรับ API key จาก https://spoonacular.com/food-api
2. เขียนโปรแกรมด้วย HTML และ JavaScript เพื่อเรียกบริการ Search Recipes เพื่อหาข้อมูลของอาหารโดยใช้ฟังก์ชัน Fetch นักศึกษาต้องศึกษาการใช้บริการเว็บเซอร์วิสจาก https://spoonacular.com/food-api/docs#Search-Recipes-Complex และ https://spoonacular.com/food-api/docs#Authentication
3. หลังจาการค้นหา โปรแกรมจะแสดงผลลัพธ์จากการค้นหาตามรูปด้านล่าง หมายเหตุ ผลการค้นหาในรูปจะเป็นการค้นหาโดยใช้คำว่า Chicken <br/>
